Overview

Applied work

This CIT4420 final assignment tackles two practical problems. Part I, Tarjan (the Thoughtful Planner), optimizes routes across a city with multiple transport modes and destinations. Part II, File Organizer, automates file categorization and cleanup in a directory.

Tarjan Planner models travel as a graph and optimizes for shortest time, lowest cost, or minimal transfers, with visual output. File Organizer focuses on automation, sorting files by type into structured folders to reduce manual work.
