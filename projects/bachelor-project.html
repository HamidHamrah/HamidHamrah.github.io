Overview

Product delivery

Brannhjelp is a bachelor project for Ignist, which needed a searchable knowledge library for customers. The solution is a web application that lets customers find guides and information, with plans for future paid access and API integration with the existing STRGI system.

The team built a full-stack app with admin and user roles, CRUD for articles and sub-articles, user management, and password reset. UX was iterated with Ignist, and the project includes testing and bug fixing.
