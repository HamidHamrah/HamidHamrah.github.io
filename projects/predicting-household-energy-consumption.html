Back to Projects GitHub Download report
Overview

Forecasting with data

This project investigates short-term household electricity consumption forecasting. The report uses an April 2025 dataset with 90,000 records and features such as temperature, household size, air-conditioning usage, and peak-hour consumption to study usage patterns.

Several statistical learning models, including linear regression, are evaluated with resampling methods like k-fold cross-validation. The analysis ties predictive accuracy to better energy planning and more sustainable distribution.
