Overview

What Simpleperf covers

Simpleperf is a simplified version of iperf built to measure network performance. The report explains why bandwidth and throughput testing matter and how streaming data between two endpoints can reveal connection quality.

The project implements a Python client and server that run timed tests and report total data transferred and bandwidth. Command-line options allow configuring host, port, and output format so tests can be repeated consistently in lab setups.
